# Fake News Detection System

## Overview
A machine learning project that classifies news as fake or real using NLP techniques.

## Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy

## Features
- Text preprocessing
- Model training (Logistic Regression, Random Forest)
- Prediction system

## How to Run
1. Install dependencies
2. Run train_model.py
3. Run predict.py

## Output
The model predicts whether a given news article is Fake or Real.

Dataset and model files are not included due to size constraints.
